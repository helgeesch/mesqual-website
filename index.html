The Pythonic framework for energy scenario analysis

A modular library for comparing energy market scenarios with powerful quantitative and qualitative analysis tools.

import pypsa
from mesqual import StudyManager
from mesqual_pypsa import PyPSADataset

# Initialize study with multiple scenarios
study = StudyManager.factory_from_scenarios(
    scenarios=[
        PyPSADataset(pypsa.Network('base.nc'), name='base'),
        PyPSADataset(pypsa.Network('scen1.nc'), name='scen1'),
    ],
    comparisons=[("scen1", "base")],
)

# Access data for all scenarios
df_prices = study.scen.fetch("buses_t.marginal_price")

# Access comparison data (deltas)
df_price_deltas = study.comp.fetch("buses_t.marginal_price")
                    

from mesqual_plexos import PlexosDataset

# Initialize dataset from PLEXOS output
dataset = PlexosDataset.from_xml_and_solution_zip(
   model='path/to/my_plexos_model.xml',
   solution='path/to/my_plexos_solution.zip',
   name='my_plexos_run',
)

# Fetch data as a DataFrame
df_prices = dataset.fetch("ST.Node.Price")
df_nodes = dataset.fetch("Node.Model")
                    

from mesqual.kpi import KPICalculator
from mesqual.kpi.library import system_cost

# Get a dataset from your study
base_scen = study.scen.get_dataset('base')

# Initialize KPI Calculator
kpi_calc = KPICalculator(dataset=base_scen)

# Calculate a pre-defined KPI
total_cost = kpi_calc.calculate_kpi(system_cost)

print(total_cost)
# > 123456789.12 EUR

What is MESQUAL?

MESQUAL provides a unified interface for handling energy systems data across any modeling platform. It's designed for efficient data handling in multi-scenario studies, featuring built-in comparison capabilities, an extensible KPI framework, and interactive visualizations.

Unified Interface

Handle data from PyPSA, PLEXOS, or custom sources with a consistent API.

Powerful Analytics

Use the extensible KPI framework and data transformation tools for deep insights.

Scenario Comparison

Natively compare scenarios to calculate deltas and analyze changes effectively.

Modular & Extensible

Integrate study-specific modules, custom data sources, and new variables with ease.
